# BankingClient

A modern Java 17 client for performing bank account transfers via a REST API.  

This project demonstrates best practices including:  
- Java 17 features (records, `HttpClient`)  
- JSON serialization/deserialization with Jackson  
- Logging using `java.util.logging`  
- Proper configuration and error handling  
- Unit and integration tests with JUnit 5 and WireMock  
- Maven build and dependency management
